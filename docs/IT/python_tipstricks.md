### Strings
cf [twitter](https://twitter.com/driscollis/status/1513442789358551040?s=20&t=UsCK2CabZtB37pN0TBvuqA)

``` python
>>> import string

>>> string.ascii_letters
'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ

>>> string.digits
'0123456789'

>>> string.hexdigits
'0123456789abcdefABCDEF'

>>> string.punctuation
'!"#$%&\'()*+,-./:;<=>?@[\\]^_`{|}~'
```
